2048 Unblock

2048 Unblock - added a little twist to a 2048 game with a non-movable circular tile

How to play:

Swipe (Up, Down, Left or Right) to move the square tiles. When two tiles with the same number touch, they merge into one.

Circular number tile will not move until it merges with the same number square tile.

When circular tile merges with square tile it will change into square tile.
